Inserta un botón de contacto por WhatsApp en tu web.

== Description ==

Configura tu número de ‘WhatsApp’ o ‘WhatsApp Business’ y deja que tus visitantes contacten contigo fácilmente.
Importante: Previamente necesitas tener instaldo y activo el plugin ACF PRO.

== Frequently Asked Questions ==

= ¿Cómo usarlo? =
Una vez instalado el plugin todas las funciones de whatsapp se reflejarán en el panel de opciones "Promaker Chat" del menú de administración.
= ¿Cuales son los requisitos? =
Previamente necesitas tener instaldo y activo el plugin ACF PRO.
